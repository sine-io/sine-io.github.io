# OPC 观察控制台需求

## Problem Frame

当前的 `OPC` 页面更像一个静态专题占位页。它能告诉读者“这里有 OPC 内容”，但不能快速回答三个更重要的问题：

- 最近变了什么
- 这意味着什么
- 接下来应该看哪里

本次需求的目标，不是把 `OPC` 扩成更大的资料页，而是把它定义成一个**阶段优先的观察控制台**。它首先服务回来看站点的“未来的自己”，帮助用户快速接上上下文；同时也要让第一次进入的读者能迅速理解页面结构和继续浏览的路径。

```text
访客进入 /projects/opc
  -> 先看 Now 区块
     -> 当前阶段
     -> 当前判断
     -> 最近变化
  -> 再扫阶段时间线
     -> 进入某个阶段
     -> 读代表性信号
     -> 沿信号跳去 Topics / Terms / Cases
```

## Requirements

**页面定位**
- R1. `/projects/opc` 必须表现为 OPC 观察控制台，而不是通用项目详情页、纯术语页或更新日志页。
- R2. 第一屏必须让回访用户在短时间内回答“现在处于哪个阶段”和“最近发生了什么”。
- R3. 第一版必须包含一个紧凑的 `Now` 区块，至少呈现当前阶段、全局当前判断和最近变化。

**时间线与判断**
- R4. 第一版时间线必须以阶段为主，而不是以离散事件为主。
- R5. 第一版必须展示 3 到 5 个顶层阶段。
- R6. 每个阶段必须包含 1 到 3 条代表性信号。
- R7. 每条信号必须同时表达三件事：发生了什么、这意味着什么、接下来去哪里看。
- R8. 页面必须同时提供一个全局当前判断，以及关键阶段上的局部判断。
- R9. 每个主要阶段都必须能引导到至少一个后续目标，不能只停留在展示层。

**扇出层**
- R10. 第一版必须提供 `Topics`、`Terms`、`Cases` 三类扇出目标，并且都能从页面内到达。
- R11. `Topics` 是第一版最完整的扇出层，必须承载主要的继续阅读路径。
- R12. `Terms` 在第一版的职责是统一语境，而不是追求术语收录的完整性。
- R13. `Cases` 在第一版的职责是提供证据和例子，而不是形成独立内容体系。

**发布门槛**
- R14. 第一版内容覆盖必须达到中等完整度，而不是空壳 MVP。
- R15. 第一版至少包含 4 到 5 个阶段、约 5 个 Topics、核心 Terms，以及 2 到 3 个 Cases。
- R16. 第一版必须让首次访问者看得懂结构，同时让回访者能快速恢复上下文，不允许只偏向其中一类用户。

## Success Criteria

- 用户进入 `OPC` 页面后，可以在 10 秒内说出当前阶段和近期变化。
- 用户无需先读长段背景说明，就能从首屏进入时间线或 `Now` 区块继续浏览。
- 至少一条完整路径成立：阶段 -> 信号 -> Topic，并且这条路径对第一次访问者也清晰。
- `Topics` 成为第一版最自然的继续阅读入口，`Terms` 和 `Cases` 成为理解与证据支持层，而不是抢主轴。
- 页面整体感知更接近“观察控制台”，而不是“专题资料页”或“更新列表”。

## Scope Boundaries

- 不把第一版做成事件优先的更新日志。
- 不要求第一版把 `Topics`、`Terms`、`Cases` 拆成独立页面。
- 不要求第一版把 `Terms` 扩成完整百科式术语库。
- 不要求第一版建立独立的 Cases 内容体系。
- 不要求第一版改造站点整体导航心智或新建新的对外入口路径。

## Key Decisions

- **阶段优先，不是事件优先**：第一版以 3 到 5 个阶段作为主骨架，先保证“现在在哪”清楚，再补更多细粒度事件。
- **判断采用混合模式**：页面有一个全局当前判断，关键阶段允许带局部判断。
- **`Now` 必须独立出现**：当前状态不能只埋在 hero 介绍里，要作为快速接续上下文的专门区块。
- **`Topics` 先最完整**：第一版的主要阅读深挖方向是专题，不是术语表，也不是案例索引。
- **信号必须可行动**：信号不能只是观察句，必须同时说明含义和下一步去向。

## Dependencies / Assumptions

- 现有站点已经有稳定的 `OPC` 入口和相关写作内容，可作为第一版的连接目标。
- 第一版的内容整理方式以人工策展为主，不依赖自动聚合或外部数据源。
- 第一版的 Cases 可以依赖现有写作内容来承担证据层，而不必新增独立文章生产流程。

## Alternatives Considered

- **事件优先时间线**：观察感更强，但第一版更容易滑向“更新流”，削弱阶段定位。
- **只有全局判断，没有局部判断**：结构更轻，但阶段变化为什么重要会变得太模糊。
- **三块扇出层平均发力**：看起来更平衡，但第一版会缺少真正的主轴。
